Improvement of a Heat Pump Monitoring and Data Acquisition System

This project involved modifying a commercial aerothermal heat pump to enhance its monitoring and data acquisition capabilities.

  • Improved sensor system in an aerothermal heat pump.
  • Implemented analog and Modbus RTU sensors on a Siemens PLC S7-1500.
  • Developed data acquisition and monitoring system in TIA Portal v17.

The following video demonstrates the monitoring system in action on a Siemens HMI screen.

Data can be accessed via the PLC web server, allowing CSV file downloads of stored data.
